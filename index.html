Skala Frekuensi / Likelihood (L)

Likelihood menunjukkan seberapa sering atau besar kemungkinan suatu insiden atau kecelakaan dapat terjadi dari sumber bahaya pada aktivitas kerja.

1
Jarang (Rare)
Kejadian sangat jarang terjadi, hanya pada keadaan ekstrem. Tidak pernah terjadi sebelumnya di fasilitas ini.
2
Kemungkinan Kecil (Unlikely)
Bisa muncul atau terjadi sewaktu-waktu. Pernah terjadi namun sangat jarang dalam 5 tahun terakhir.
3
Mungkin (Possible)
Seharusnya bisa terjadi atau mungkin telah terjadi sewaktu-waktu. Terjadi sekali atau lebih dalam setahun.
4
Kemungkinan Besar (Likely)
Dapat terjadi dengan mudah dan muncul dalam banyak keadaan. Terjadi beberapa kali dalam setahun.
5
Pasti (Certain)
Kejadian berulang-ulang, hampir muncul pada semua keadaan. Terjadi setiap hari atau setiap minggu kerja.
⚡
Skala Keparahan / Severity (S)

Severity menunjukkan tingkat kerusakan atau kerugian yang ditimbulkan apabila insiden atau kecelakaan terjadi dari sumber bahaya tersebut.

1
Tidak Signifikan (Insignificant)
Tidak menimbulkan cedera pada pekerja; kerugian aset atau material sangat kecil dan dapat diabaikan.
2
Kecil (Minor)
Cedera ringan yang memerlukan P3K; tidak ada kehilangan hari kerja; kerugian aset bersifat kecil dan dapat diperbaiki.
3
Sedang (Moderate)
Cedera berat yang memerlukan perawatan medis profesional; kehilangan beberapa hari kerja; kerugian aset sedang.
4
Berat (Major)
Cedera parah atau cacat permanen; kehilangan banyak hari kerja; kerugian aset besar dan mengganggu operasional.
5
Bencana (Catastrophic)
Mengakibatkan kematian satu atau lebih pekerja; kerugian aset sangat parah; operasional terhenti sepenuhnya.
🎯
Matriks Risk Index (L × S)

Risk Index diperoleh dari perkalian nilai Likelihood (L) dan Severity (S). Hasilnya menentukan prioritas tindakan pengendalian yang harus segera dilakukan.

1 – 4
LOW — Risiko Rendah
Risiko dapat diterima. Pertahankan kondisi yang ada; lakukan pengawasan berkala; dokumentasikan temuan.
5 – 10
MEDIUM — Risiko Sedang
Perlu pengendalian tambahan dalam waktu yang direncanakan. Terapkan pengendalian administratif dan APD yang sesuai.
11 – 16
HIGH — Risiko Tinggi
Pengendalian segera diperlukan. Pertimbangkan penghentian sementara kegiatan hingga risiko dapat dikendalikan secara efektif.
17 – 25
EXTREME — Risiko Sangat Tinggi
Hentikan kegiatan segera. Diperlukan tindakan menyeluruh sebelum aktivitas dapat dilanjutkan kembali.
Sumber: AS/NZS 4360:2004 Risk Management Standard. Matriks digunakan untuk metode semi-kualitatif HIRARC.

Sinkronisasi & Database Real-Time
🔌 Belum terhubung · Semua perubahan hanya tersimpan lokal
🔑 Kode Room Kolaborasi
Bagikan kode ini ke rekan agar terhubung ke room yang sama. Semua perubahan (titik, warna, GeoJSON) akan real-time.
⚙️ Konfigurasi Firebase
Isi untuk mengaktifkan sinkronisasi GeoTIFF besar (1 MB–3 GB+)
📋 Cara Setup Firebase (GRATIS) ▾
Realtime Database (fasilitas, warna, GeoJSON):
1. console.firebase.google.com → Buat project
2. Build → Realtime Database → Create database
3. Rules tab → Edit: ".read": true, ".write": true → Publish
4. Project Settings → General → copy Web API Key & Project ID

Firebase Storage (GeoTIFF hingga 3 GB+):
5. Build → Storage → Get started
6. Rules tab → Edit: allow read, write: if true; → Publish
7. Copy bucket URL (format: project-id.appspot.com)

⚠️ Perhatian: Rules publik aman untuk riset/skripsi, namun ganti ke rules yang lebih ketat untuk produksi.
🛰️ File GeoTIFF di Cloud Storage
File GeoTIFF yang diupload ke room ini dapat dimuat oleh semua pengguna room yang sama. Upload otomatis ketika GeoTIFF dimuat ke peta dan sinkronisasi aktif.
Storage Terpakai —
Belum ada GeoTIFF di cloud room ini
💡 Muat GeoTIFF ke peta saat sinkronisasi aktif → file otomatis diupload ke Firebase Storage → rekan tim dapat mengunduh dari daftar di atas.
ℹ️ Cara kerja sinkronisasi:
• Hubungkan → semua perubahan (tambah, edit, hapus fasilitas, aktivitas, GeoJSON) langsung disinkronkan ke server.
• Putuskan & Hapus Konfigurasi → memutus koneksi secara permanen. Halaman yang di-reload tidak akan reconnect otomatis.
• Untuk menyambung kembali, isi ulang form di atas dan klik Hubungkan.
